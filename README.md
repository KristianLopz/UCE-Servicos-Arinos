# UCE - Serviços Arinos

Projeto frontend para o sistema "Serviços Arinos", desenvolvido como parte de um trabalho acadêmico no Instituto Federal do Norte de Minas Gerais (IFNMG).

Descrição
- Frontend em React + Vite com componentes reutilizáveis e estilos Tailwind/CSS.
- Estrutura de pastas organizada em `src/app`, `src/components`, `src/pages`, `src/styles`.
- Mock data e utilitários para formatação e uso local.

Como executar
1. Instale as dependências: `pnpm install` (ou `npm install`/`yarn` conforme preferência).
2. Rode o servidor de desenvolvimento: `pnpm dev`.

Observações
- Este upload substitui os arquivos existentes no repositório remoto conforme solicitado.
- Trabalho realizado como parte de atividades do IFNMG.
